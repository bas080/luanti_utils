Module emerge_node

Ensure a node is loaded and call a callback with it.

This function attempts to get the node at pos. If the node is not loaded yet, it requests the server to emerge the area containing pos. Once the node becomes available, the provided callback is called with the node and its position.

The callback is always called asynchronously if the node needs to be emerged. Otherwise, it may be called immediately.
